# 📊 Data Cleaning & Dashboard Project (Pandas + Power BI)

This project demonstrates an end-to-end data workflow — starting from **data cleaning using Pandas (Python)** to building an **interactive dashboard in Power BI**.

The goal is to transform raw data into meaningful insights and present them visually for better decision-making.

---

## 🚀 Project Workflow

1. **Data Collection**
   - Raw dataset in CSV format

2. **Data Cleaning (Python - Pandas)**
   - Removed missing values
   - Handled duplicates
   - Standardized column names
   - Converted data types
   - Prepared dataset for analysis

3. **Data Visualization (Power BI)**
   - Imported cleaned dataset
   - Created interactive dashboard
   - Built charts and KPIs
   - Generated insights

---

## 🛠️ Tools & Technologies

- Python 🐍
- Pandas 📊
- Power BI 📈

---

## 🧹 Data Cleaning Steps (Pandas)

- Used `dropna()` / `fillna()` to handle missing values  
- Removed duplicate rows using `drop_duplicates()`  
- Cleaned column names (`lowercase`, `strip`)  
- Converted columns to appropriate data types  
- Filtered and structured the dataset  

---

## 📊 Dashboard Features (Power BI)

- 📌 KPI indicators (e.g., total sales, count, etc.)
- 📌 Bar charts & pie charts
- 📌 Trend analysis (time-based)
- 📌 Filters & slicers for interactivity
